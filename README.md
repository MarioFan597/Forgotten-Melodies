A mod that provides new items and a dungeon to explore: using a musical instrument, you are to traverse the forgotten Resonant Ruin and obtain the remaining pieces of an ancient machine.

## âœ¨ Gameplay Features

- **Melody Workbench**: A custom workbench to craft an instrument and an ancient machine 
- **Kantele**: An instrument that can be manually played or upgraded to play melodies that have magical properties. It can change states of certain blocks or plants within the Resonant Ruin
- **Resonant Ruin**: An obris fragment that leads to a forgotten temple, where parts of an ancient machine resides.
- **Elemental Resonator**: A machine that echoes an elemental fragment's magical powers to provide buffs. 
- **Puzzle Blocks**: Blocks that change states when the correct melody is played.
